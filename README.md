## 📖 মাঙ্গা সম্পর্কে

**Tonikaku Kawaii** (とにかくかわいい — "যাই হোক না কেন, সে অসাধারণ সুন্দর") হলো **Kenjiro Hata** রচিত একটি রোমান্টিক কমেডি মাঙ্গা। এটি **Shogakukan**-এর *Weekly Shonen Sunday* পত্রিকায় প্রকাশিত হয়েছিল।

| বিষয় | তথ্য |
|-------|------|
| 📝 লেখক | Kenjiro Hata |
| 🏢 Publisher | Shogakukan |
| 📅 শুরু | January 2018 |
| ✅ Status | Completed |
| 📚 Genre | Romance, Comedy, Slice of Life |
| 🔞 Rating | Shounen |

---

## 📋 সারসংক্ষেপ

Nasa Yuzaki — যে একটি ট্রাক দুর্ঘটনায় বেঁচে যায় এক রহস্যময় মেয়ের কারণে। সেই মেয়ে, **Tsukasa**, তাকে একটি শর্তে বিয়ে করতে রাজি হয়। তাদের নবদম্পতি জীবনের মজাদার ও মিষ্টি গল্পই এই মাঙ্গার মূল বিষয়।
